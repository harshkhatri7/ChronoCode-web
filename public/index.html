How to use ChronoCode

ChronoCode runs in the background and automatically records your file changes on save.

1

Open this folder in your IDE

Launch VS Code, Cursor, Zed, or any editor on this repository.

2

Make changes & save files

Modify any code file (e.g. edit comments or code) and hit save.

3

Observe the Timeline

A snapshot node will appear on the timeline below. Click it to inspect diffs or restore!
